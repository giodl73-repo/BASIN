# Water Peer Panel (illustrative)

Archetype peer reviewers for BASIN research outputs — distinct from the parliament
(adversarial design review) and the editorial gate (form review). Each is an
archetype, not a named person.

| ID | Reviewer archetype | Expertise |
|---|---|---|
| R-P1 | Water-resources planner | Basin planning, supply-demand balance, multi-scale systems |
| R-P2 | Hydrologist | Surface/groundwater hydrology, drought, firm yield |
| R-P3 | Water economist | Benefit-cost, pricing, demand management |
| R-P4 | Network / systems engineer | Conveyance graphs, hydraulic networks, optimization |
| R-P5 | Water-law / governance scholar | Rights, compacts, treaties, transboundary governance |

Assignment guide:

- Conveyance / network method → R-P4, R-P1
- Supply / drought / yield → R-P2, R-P1
- Benefit-cost / pricing → R-P3
- Rights / transboundary / governance → R-P5, R-P1
- Tier/SLA + multi-scale methodology → R-P1, R-P4, R-P5
