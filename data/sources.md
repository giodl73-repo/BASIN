# Source Registry

Every cited corpus quantity points to a source id in this registry. Entries can
be expanded as the corpus grows; a source id is stable once cited. Candidate
public sources for the water-supply corpus include USGS National Water
Information System, USBR reservoir and project data, state water-resources
agency reports, EPA SDWIS public water systems, and Census/ACS service-area data.
