# BASIN Open Adoption Guide

## Purpose

BASIN is public and open to use as a reference model for evidence-gated water
supply, conveyance, resilience, reuse, and access analysis. Its next public
milestone is a narrow, source-backed first cited run.

Public use does not create a hydraulic model of record, water-rights
determination, engineering study, environmental review, project selection, or
agency, basin-authority, tribal, or international-body endorsement.

## First Run Path

Use [`first-public-run-worksheet.md`](first-public-run-worksheet.md) to scope
one basin, utility service area, conveyance path, drought scenario, reuse
question, or access gap.

The first run should prove the source-to-corpus-to-gap workflow, not settle a
water allocation or engineering case.

## Contribution Targets

- public source inventories for supply, demand, storage, conveyance, drought,
  reuse, leakage, rights, or access evidence;
- source/claim corrections for hydrologic, demand, rights, or reliability
  language;
- reviewer notes from water planning, hydrology, rights, utility operations,
  resilience, tribal/jurisdictional, finance, or public-language lenses.

## Gate

Decision: **open_for_first_public_run_scoping**

Rationale: BASIN can accept bounded source and review contributions before a
first cited run. Reuse alone does not create engineering, rights, environmental,
project-selection, endorsement, or validation claims.
